<h1>Customer Invoice Project</h1>

 

<h2>Description</h2>
In this project, we will build a NoSQL Document-Oriented database using CouchDB. The database will be used to provide content to a website app, which will display customer invoices of an online retailer and the items on the invoices.

<br />


<h2>Languages and Utilities Used</h2>

- <b>JavaScript</b> 
- <b>MapReduce</b>
- <b>Fauxton</b>

<h2>Environments Used </h2>

- <b>Apache CouchDB</b>
